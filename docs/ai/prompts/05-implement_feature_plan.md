# Implement Feature

Implement the approved feature.

Before beginning:

1. Read AGENTS.md.
2. Restore project context.
3. Review the feature plan.

During implementation:

- Follow project standards.
- Maintain consistency with the existing architecture.
- Write clean, maintainable code.
- Add documentation where appropriate.
- Add tests when applicable.
- Keep changes focused on the requested feature.

Avoid unrelated refactoring unless necessary.

When complete:

- Update project memory.
- Record architectural decisions.
- Update implementation status.
- Recommend next actions.
